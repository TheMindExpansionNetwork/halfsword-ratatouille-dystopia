## Visual Props Pak (Demo)

This folder contains `RatatouilleVisuals_Demo.pak` — a starter visual override that adds more cooking utensils (pans, boards, plates, etc.) into the game world as props.

- Installed automatically by `install.ps1` / `install.bat` into `HalfswordUE5/Content/Paks/`
- Demonstrates the full asset replacement pipeline
- Full custom weapon models (frying pans as swords/axes, rolling pins, cleavers, pot barrels, etc.) are in active development using the autonomous factory

Drop any additional .pak files here for more kitchen chaos.

See the main README for install and console commands (`KitchenDystopia` etc.).

*Part of Ratatouille: Dystopia Edition v0.3+*
